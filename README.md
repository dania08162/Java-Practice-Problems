## Student Managment System
- Lets the user add student records, view all students, and delete student records.
- Uses a menu-driven interface with a switch statement to navigate between options.
- Stores student information (Student ID and Name) in a text file for permanent storage.
- Uses file handling to save, read, and update records.
- Technologies used: Java, Apache NetBeans, File Handling (FileReader, FileWriter, BufferedReader, PrintWriter), Scanner Class.
