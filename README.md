# Java-Practice-Problems
Java Practice Problems 

## Java Diner
- Lets the user choose meal type, select something from the menu, and finally pay.
- Uses switch
- Technologies used: Java, IntelliJ

## Student Managment System
- Lets the user add student records, view all students, and delete student records.
- Uses a menu-driven interface with a switch statement to navigate between options.
- Stores student information (Student ID and Name) in a text file for permanent storage.
- Uses file handling to save, read, and update records.
- Technologies used: Java, Apache NetBeans, File Handling (FileReader, FileWriter, BufferedReader, PrintWriter), Scanner Class.

## Smart Parcel Delivery System
- Allows users to manage and calculate shipping details for different package types.
- Supports Standard, Fragile, Express, and Cold Storage packages.
- Uses inheritance, polymorphism, constructors, arrays, method overriding, and encapsulation.
- Calculates shipping costs based on package type, weight, destination zone, and special delivery requirements.
- Estimates delivery times automatically for each package type.
- Generates a complete delivery report displaying package information, shipping costs, and estimated delivery days.
- Technologies used: Java, Apache NetBeans, Object-Oriented Programming (OOP).
